## 📂 Folder Structure

- `app/` - Next.js pages
- `components/` - Reusable components
- `lib/` - Utilities & helpers
- `styles/` - Global styles
- `database/` - Database schema
- `public/` - Static assets

## 🗄️ Database Setup

1. Create project di https://supabase.com
2. Copy database/schema.sql
3. Paste ke Supabase SQL Editor
4. Copy Project URL & Anon Key ke .env.local

## 🎯 Fitur

- Authentication (Login/Register)
- 3 Platform Dashboard (Karyawan, Owner, Customer)
- Order Management
- PDF Invoice (Paperless)
- Reporting & Analytics
- WhatsApp Integration (Button only)
